*** Essa aplicação é para fins de aprendizado em Front End! ***

Essa aplicação conecta-se a API DOG que gera imagens aleatorias de doguinhos, como essa API não gera os nomes e historias dos animais, foi criado um arquivo JSON com informações do site "https://www.amigonaosecompra.com.br/", então as informações que aparecem ao lado da foto dos bichinhos não são de fato deles.

A aplicação é intuitiva onde o botão "próximo" gera informações e imagens de outro animal, click no botão adotar para garantir seu amiguinho.
